# nlb-digit-test
NLBDigit practical interview test project

## Task:

Implement a Basic Payment Transfer Service
Objective: Develop a simple payment transfer feature for a digital banking platform that ensures secure and efficient fund transfers between accounts, focusing on robust functionality and high code quality.
 
## Requirements

Feature Overview: Create a service enabling users to transfer funds between accounts within the same banking platform.

## Functional Requirements:

- ✅ Users should be able to initiate a transfer by providing both the source and destination account IDs, along with the transfer amount.
- ✅ The system must validate that the source account has sufficient funds before processing the transfer.
- ✅ Implement comprehensive error handling to ensure reliability and security.
- ✅ Record each successful transaction accurately for audit and tracking purposes.
 
☑️🚧⚠️ Please use TCA (The Composable Architecture)
